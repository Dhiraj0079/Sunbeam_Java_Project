1. Who developed the Java programming language?
A) Microsoft
B) Sun Microsystems
C) Google
D) IBM
Answer: B) Sun Microsystems

2. In which year was Java first released?
A) 1991
B) 1995
C) 2000
D) 2005
Answer: B) 1995

3. Java is best described as which type of language?
A) Low-level language
B) Machine-dependent language
C) Platform-independent language
D) Assembly language
Answer: C) Platform-independent language

4. Which slogan is commonly associated with Java?
A) Code Fast, Run Faster
B) Write Once, Run Anywhere
C) Build Once, Compile Everywhere
D) Code Once, Debug Forever
Answer: B) Write Once, Run Anywhere

5. Which component of Java makes it platform independent?
A) JDK
B) JRE
C) JVM
D) JIT
Answer: C) JVM
